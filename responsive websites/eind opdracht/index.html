Over mij

Mijn naam is Leroy Verwey, ik ben 19 jaar en ben geboren in 's-Hertogenbosch op 3 november 2006. Tijdens mijn 1e jaar van mijn opleiding software Developer heb ik ervaring opgedaan met verschillende programmeertalen en technieken, zoals HTML, CSS, JavaScript, Bootstrap, Php en C#.
Daarnaast heb ik gewerkt aan diverse projecten waarin ik heb geleerd om zowel zelfstandig als in teamverband te werken. Hierbij heb ik niet alleen mijn technische vaardigheden ontwikkeld, maar ook mijn communicatie vaardigheden.

Waarom deze website?

Deze website heb ik gemaakt voor mijn eindopdracht van periode 2 voor het vak Interactieve Responsive Website. Ik vind dit een leuke manier om mij voor te stellen aan mensen, waarbij ze zelf kunnen kiezen wat ze willen lezen.
